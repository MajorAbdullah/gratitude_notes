# Logo Assets Guide

## Logo Requirements

Place your logo files in this directory with the following specifications:

### App Icon (Required)
- **logo.png** - 1024x1024 pixels, PNG with transparency
- This will be used to generate all required icon sizes

### Onboarding/Splash (Optional)
- **logo_onboarding.png** - 512x512 pixels, PNG with transparency
- Used in onboarding screens and splash screen

## Logo Design Suggestions for Gratitude Notes

### Theme
- Warm, peaceful, positive
- Colors: Purple (#6B4EFF), Gold/Yellow (gratitude), White
- Style: Modern, minimal, clean

### Symbol Ideas
- Heart with a pen/pencil
- Open journal/notebook with a heart
- Three dots/circles (representing 3 gratitude notes)
- Sunrise/sunset (new day, reflection)
- Hands in gratitude gesture
- Simple "G" monogram with decorative elements

### Design Principles
- Should look good at small sizes (16x16)
- Should work in both light and dark modes
- Avoid too much detail
- Use 2-3 colors maximum

## Icon Sizes Needed (Auto-generated by flutter_launcher_icons)

The plugin will generate these automatically from logo.png:
- mipmap-mdpi: 48x48
- mipmap-hdpi: 72x72
- mipmap-xhdpi: 96x96
- mipmap-xxhdpi: 144x144
- mipmap-xxxhdpi: 192x192
